There are three conditions where we are silently dropping routes:

1. If route endpoints isn't >= minEndpointsByZone
2. If route does not have any endpoints
3. If route does not have any endpoints for a zone

We should add all routes, and if routes endpoints match the condition  

then we filter the endpoints to only get <code>zone</code> endpoints only.

First commit is a reproducer test case (pipeline should fail)
