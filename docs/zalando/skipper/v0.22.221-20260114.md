context: span tag <code>shadow: true</code> should only be set for shadow traffic (<a class="issue-link js-issue-link" data-error-text="Failed to load title" data-id="3808252291" data-permission-text="Title is private" data-url="https://github.com/zalando/skipper/issues/3825" data-hovercard-type="pull_request" data-hovercard-url="/zalando/skipper/pull/3825/hovercard" href="https://github.com/zalando/skipper/pull/3825">#3825</a>)

Shadow Traffic usually means the duplicated traffic whose response is  

discarded. The loopbackInternal method get called for teeLoopback filter  

and loopbackIfStatus filters. At the moment in both the cases we tag the  

span with <code>shadow: true</code>. We wanna do this only for the teeLoopback case  

and this change accomplishes that.
