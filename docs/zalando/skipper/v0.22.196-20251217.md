> v6.0.0
>
> v6 - What's new [!IMPORTANT]
> > actions/upload-artifact@v6 now runs on Node.js 24 (<code>runs.using: node24</code>) and requires a minimum Actions Runner version of 2.327.1.
> > If you are using self-hosted runners, ensure they are updated before
> > upgrading.
>
> Node.js 24
>
> This release updates the runtime to Node.js 24. v5 had preliminary
> support for Node.js 24, however this action was by default still running
> on Node.js 20. Now this action by default will run on Node.js 24.
>
