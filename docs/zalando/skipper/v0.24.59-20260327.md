## changes
Changes

Extend predicate <a href="https://opensource.zalando.com/skipper/reference/predicates/#otelbaggage" rel="nofollow">OTelBaggage</a> to match with value

Context

Currently the predicate OTelBaggage only matches based on key.

This needs to be extended to match based on value if the value is not  

empty.

Tasks

* [x] Update <code>baggagePredicate</code> struct to include optional value field
* [x] Modify <code>Create()</code> to accept 1 or 2 arguments (key only or  
  key+value)
* [x] Update <code>Match()</code> to check value when provided, key-only when not
* [x] Add test cases for value matching scenarios
* [x] Run tests to verify working implementation
