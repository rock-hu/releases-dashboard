## changes
10.3.13 - May 2026

* (PW-3185) <code>MtSwiftMessage.toJson()</code> now uses 1-based months (January=1) for Calendar fields and emits a <code>schemaVersion</code> marker. <code>fromJson()</code> reads both new and legacy (0-based) payloads transparently
* Migrated XML escape/unescape from deprecated <code>org.apache.commons.lang3.StringEscapeUtils</code> to <code>org.apache.commons.text.StringEscapeUtils</code>. Adds new runtime dependency on <code>org.apache.commons:commons-text:1.15.0</code>
* Updated gson from 2.13.2 to 2.14.0
