## changes
Announcement

Spring Cloud Open Service Broker 5.0.0-RC2 is now available. Version 5.0.0 is built against Spring Boot 4.0 and Jackson 3, and requires Java 17. It includes support for OSB API v2.16.

What's Changed

* Use Spring Boot managed version of JSpecify
* Use api scope for JSpecify dependency to be consistent with other Spring Projects and follow JSpecify guidance
