Android specific

* Fix included-build <code>gradle.properties</code> resolution for monorepos by reading from parent Gradle start parameters (<code>projectDir</code>/<code>currentDir</code>) before falling back to <code>../../android/gradle.properties</code>, so <code>hermesV1Enabled</code> can be inherited correctly. (<a href="https://github.com/facebook/react-native/commit/c5ae0c820a571141a43de980cb011d232fc2268f">c5ae0c820a</a> by <a href="https://github.com/Titozzz">@Titozzz</a>)
* Provide symbol fallbacks for <code>inspector-modern/chrome/Registration.h</code> when HermesV1 is disabled. (<a href="https://github.com/facebook/react-native/commit/5ee695ad8f41d7c9a92a83fa1f0acd227cdab7b8">5ee695ad8f</a> by <a href="https://github.com/tjzel">@tjzel</a>)
* ReactPlugin.kt always setting <code>hermesV1Enabled</code> to <code>true</code> (<a href="https://github.com/facebook/react-native/commit/45896674d14d4b0104cb166e83c5b403f90981c0">45896674d1</a> by <a href="https://github.com/tjzel">@tjzel</a>)
