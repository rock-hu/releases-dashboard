# 0.84.0-rc.3

## release on 20260127
## description
## changes
Added

* <strong>Animated</strong>: Added <code>transformOrigin</code> prop support to the <code>AnimationBackend</code>. (<a href="https://github.com/facebook/react-native/commit/20779edd3bc9f1c205a063e15b324466ee509c24">20779edd3b</a> by <a href="https://github.com/coado">@coado</a>)
* <strong>Animated</strong>: Added support for <code>backfaceVisibility</code> prop and added missing <code>flex</code> style to the <code>AnimatedPropsBuilder</code>. (<a href="https://github.com/facebook/react-native/commit/d379cde03682923180f2cf2594a039fa3b32b326">d379cde036</a> by <a href="https://github.com/coado">@coado</a>)
* <strong>Animated</strong>: Added support for <code>borderCurves</code>, <code>borderStyles</code>, <code>pointerEvents</code>, <code>isolation</code>, <code>cursor</code>, <code>boxShadow</code>, and <code>mixBlendMode</code> props to the AnimationBackend. (<a href="https://github.com/facebook/react-native/commit/6d5b524f876a33b883128714e016052c620e1b41">6d5b524f87</a> by <a href="https://github.com/coado">@coado</a>)
* <strong>Animated</strong>: Added support for <code>overflow</code>, <code>position</code>, <code>zIndex</code>, and <code>direction</code> props to the AnimationBackend. (<a href="https://github.com/facebook/react-native/commit/bed1a9a20f8a239ad1c096b255b6dea716229074">bed1a9a20f</a> by <a href="https://github.com/coado">@coado</a>)

Fixed

iOS specific

* <strong>Codegen</strong>: Fix codegen failing when project is inside a hidden folder (<a href="https://github.com/facebook/react-native/commit/a12b442d26d0b3a47b4a3d2c1ccccfeb890b9bdc">a12b442d26</a> by <a href="https://github.com/dinisnunes1">@dinisnunes1</a>)
