# 0.83.10

## release on 20260626
## description
## changes
v0.83.10

Added

iOS specific

* Cache prebuilt iOS binaries in <code>~/Library/Caches/ReactNative</code> so Hermes, ReactNativeDependencies and ReactNativeCore tarballs are reused across clean installs and projects instead of being re-downloaded from Maven (<a href="https://github.com/react/react-native/commit/9a0b05b8d07f3468a08077119189406d56a230ba">9a0b05b8d0</a> by <a href="https://github.com/cipolleschi">@cipolleschi</a>)

Fixed

* Fixed Yoga node ownership when <code>display: contents</code> is used in absolutely positioned subtrees (<a href="https://github.com/react/react-native/commit/f2f92098ddf996d3f75cfbaa143d7151c29776f7">f2f92098dd</a> by <a href="https://github.com/j-piasecki">@j-piasecki</a>)
* Fixed <code>display: contents</code> nodes having <code>hasNewLayout</code> set incorrectly (<a href="https://github.com/react/react-native/commit/2546ce4d8219050fcd1bf432c7c830c9fd70c9af">2546ce4d82</a> by <a href="https://github.com/j-piasecki">@j-piasecki</a>)
* React Native DevTools: Fix a bug where we would incorrectly flag apps using additional Hermes runtimes (e.g. Reanimated) as being multi-host (<a href="https://github.com/react/react-native/commit/c8005032140f1aff16027b5ed53caea6c9d299f5">c800503214</a> by <a href="https://github.com/huntie">@huntie</a>)

Android specific

* <code>fetch()</code> response URL is now correct after a redirect (<a href="https://github.com/react/react-native/commit/fbe6a686e65e70dd61700413084ddc54c0b86765">fbe6a686e6</a> by <a href="https://github.com/MarkCSmith">@MarkCSmith</a>)
* Limit WebSocket queue size for the packager connection to prevent the Android inspector from being force-disconnected on large payloads (<a href="https://github.com/react/react-native/commit/7164f96d581115e6a7a5646a50ded8e5fdff7742">7164f96d58</a> by <a href="https://github.com/huntie">@huntie</a>)

iOS specific

* Make Podfile.lock SPEC CHECKSUMS deterministic across machines by sorting Dir.glob results in Yoga.podspec and using a dynamically computed Pods-relative path in hermes-engine.podspec (<a href="https://github.com/react/react-native/commit/64c9663152879a87d061a3f01b9e9c4e98cc73bc">64c9663152</a> by <a href="https://github.com/IsaacIsrael">@IsaacIsrael</a>)
* Fixes crash when changing the value of <code>removeClippedSubviews</code> (<a href="https://github.com/react/react-native/commit/91e3f773b7e571a503b57e09a1cb8a44ff26cd1e">91e3f773b7</a> by <a href="https://github.com/javache">@javache</a>)
