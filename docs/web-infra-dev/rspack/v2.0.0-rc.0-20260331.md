Highlights 💡

Support <code>import.meta.main</code>

<code>import.meta.main</code> is a boolean flag used in ES modules to determine whether the current module is the program’s entry module. It evaluates to true when the module is executed as the main entry, and to false when it is imported by another module. You can think of it as the ESM equivalent of <code>require.main === module</code> in CommonJS. It is useful for running startup logic, initialization code, or CLI behavior only when a module is launched directly.
