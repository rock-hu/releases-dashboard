更新内容

新特性 🎉

* A new additive app-tools hook <code>modifyBuilderEnvironments</code> runs in <code>generateBuilder</code> after the framework's static environment merge and before <code>createBuilder</code>. It mirrors <code>modifyEntrypoints</code> as a transform-object async hook: each tapped callback receives <code>{ environments }</code> and may return a replacement map that chains to the next callback; tapping nothing leaves the environments byte-identical, so the change is fully backward compatible. This lets frameworks programmatically add environments (e.g. a custom browser worker environment), set per-environment <code>output</code>/<code>distPath</code>/ordering, or otherwise adjust the resolved environments map instead of being limited to a config-static merge.  
  feat(app-tools): 新增 <code>modifyBuilderEnvironments</code> hook，可在创建 builder 前变换 builder 环境表
