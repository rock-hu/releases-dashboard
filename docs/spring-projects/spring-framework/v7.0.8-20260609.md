## changes
:warning: Security Fixes

This maintenance release fixes a high number of CVEs. You can learn more about this in the <a href="https://spring.io/blog/2026/06/01/spring_and_security_in_the_times_of_ai" rel="nofollow">"Spring and Security In The Times Of AI"</a> blog post. Here is the full list of 16 CVEs:

* <a href="https://spring.io/security/cve-2026-41841" rel="nofollow">CVE-2026-41841</a> "Spring Framework Information Disclosure via Static Resource Cache in Spring MVC and WebFlux"

* <a href="https://spring.io/security/cve-2026-41844" rel="nofollow">CVE-2026-41844</a> "Spring Framework Open Redirect in Spring MVC and WebFlux"

* <a href="https://spring.io/security/cve-2026-41846" rel="nofollow">CVE-2026-41846</a> "Spring Framework Cross-site Scripting via JSP Form Tags"
