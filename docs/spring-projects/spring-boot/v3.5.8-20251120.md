## changes
:warning: Noteworthy changes

* This release <a href="https://github.com/spring-projects/spring-boot/issues/48104" data-hovercard-type="issue" data-hovercard-url="/spring-projects/spring-boot/issues/48104/hovercard">contains a fix</a> to get Testcontainers working with modern Docker versions. If this causes problems in your setup, <a href="https://docs.spring.io/spring-boot/3.4/how-to/testing.html#howto.testing.testcontainers-api-downgrade" rel="nofollow">you can downgrade the minimum Docker API</a>, effectively reverting that change.
