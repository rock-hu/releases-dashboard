Quick Start

Non-HA:

    kubectl create namespace argocd
    kubectl apply -n argocd -f https://raw.githubusercontent.com/argoproj/argo-cd/v3.2.1/manifests/install.yaml

HA:

    kubectl create namespace argocd
    kubectl apply -n argocd -f https://raw.githubusercontent.com/argoproj/argo-cd/v3.2.1/manifests/ha/install.yaml

Release Signatures and Provenance

All Argo CD container images are signed by cosign. A Provenance is generated for container images and CLI binaries which meet the SLSA Level 3 specifications. See the <a href="https://argo-cd.readthedocs.io/en/stable/operator-manual/signed-release-assets" rel="nofollow">documentation</a> on how to verify.

For a detailed breakdown of the key changes and improvements in this release, check out the <a href="https://blog.argoproj.io/argo-cd-v3-0-release-candidate-a0b933f4e58f" rel="nofollow">official blog post</a>
